## Pre-alpha v5.0.0 {{  date: '2024-07-09T16:51Z' }}

### <SparkleIcon /> New Features and Technological Choices:
- **Database:** PostgreSQL usage has been confirmed. The database schema and basic configurations have been established.
- **Server-Side Technology:** Node.js has been chosen as the server-side technology. The development environment has been set up, and foundational components have been created.
- **Version Control:** Git has been selected for version control. A central Git repository has been created, and access permissions have been configured.
- **Cache Technology:** It was decided to use Redis. Redis will be used to speed up database queries and improve performance.
- **Other Technologies:**
  - **Custom Technologies:** Initial drafts of our custom-developed technologies and tools have been completed. These technologies have been optimized specifically for the project and are designed to work in harmony with the existing features.

### <SparkleIcon /> Work Completed:
- **Project Plan:** A comprehensive project plan detailing requirements and technical aspects has been created. This plan includes development processes and key milestones.
- **Initial Prototype:** Development work has begun for the initial prototype version. Basic functionalities are being worked on, and initial testing is underway.
